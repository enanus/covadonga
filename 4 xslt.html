XSLT

Es un lenguaje que permite aplicar una transformación a un documento XML para obtener otro documento XML, un documento HTML o un documento de texto plano.

La hora de estilos XSLT con las reglas de transformación es también un documento de texto XML en sí, generalmente con extensión .xsl, por lo que se podrá comprobar si está bien formada o no. El funcionamiento lo podemos observar en la siguiente imagen:

A un documento XML se le pueden aplicar una o varias transformaciones XSLT e incluso una transformación CSS. Las hojas de estilos XSLT son más útiles que las hojas de estilos CSS porque:

  • Permiten cambiar el orden de los elementos
  • Permiten realizar el orden de los elementos
  • Permiten agrupar elementos
